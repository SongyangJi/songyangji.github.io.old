快速幂与慢速乘

小栗子
快速幂： ${ {2^7=22^6=2(2^3)^2=2*(2*(2^2))^2} }$

也就是：${ {2^{2^2+2^1+2^0} } }$
时间复杂度：O(logn)

慢速乘$ { { 37=3(2^2+2^1+2^0) } } $

常见问法：
Leading and Trailing LightOJ - 1282
输出一个大数的最右面3位（%1000即可），和最左面3位。
如果要输出最左面的3位呢？
${ {n^k=10^p} }$
p应该是个带小数的浮点数m,取对数 ${ {p=k*log_{10}(n)} }$,(p的小数部分)*100取整就是答案，因为p的整数部分只控制多少位。
